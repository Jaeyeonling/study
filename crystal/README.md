# Crystal

## How to use
 
### Compiling and running at once

```shell script
crystal run ${file_name}.cr
```

### Creating an executable 
```shell script
crystal build ${file_name}.cr
```

### Release builds
```shell script
crystal build ${file_name}.cr --release
```

### Creating a Crystal project
```shell script
crystal init app ${app_name}
```
